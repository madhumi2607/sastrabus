## Overview
SASTRABUS is a real-time smart campus bus tracking system designed to improve transportation transparency and student convenience.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB
- Hardware: ESP32, Neo-6M GPS

## Features
- Real-time bus location tracking
- Live map visualization
- Route progression display
- Driver contact information
- Secure student access

## Project Status
This project currently runs locally and is not deployed.
